# Music Cutter

Десктопное приложение для вырезания фрагментов из MP3 аудиофайлов.

## Стек технологий

- **Backend:** Rust + Tauri 2.x
- **Frontend:** React 19 + TypeScript + Vite + Chakra UI
- **Аудио обработка:** FFmpeg

## Разработка

```bash
npm run tauri dev
```

## Сборка

```bash
npm run tauri build
```
